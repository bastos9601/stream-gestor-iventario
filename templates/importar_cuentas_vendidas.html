Instrucciones de Importación:
  • Selecciona un archivo .txt que haya sido exportado previamente desde esta aplicación
  • El archivo debe contener cuentas vendidas en el formato estándar
  • Las cuentas se importarán con estado "Vendida"
  • Si una cuenta ya existe (mismo email), no se importará
  • Se mantendrán todos los datos: plataforma, email, contraseña, precio, fechas, etc.
Solo archivos de texto (.txt) son permitidos
Importante:
  • Esta acción importará las cuentas como VENDIDAS
  • Las cuentas no estarán disponibles para venta
  • Se mantendrá el historial de ventas
  • Verifica que el archivo contenga la información correcta antes de importar

Lo que se importa:
  • Plataforma de streaming
  • Email de la cuenta
  • Contraseña
  • Precio de venta
  • Fecha de compra
  • Fecha de venta
  • Datos del comprador
  • Notas adicionales
Protecciones:
  • No se duplican cuentas
  • Validación de formato
  • Manejo de errores
